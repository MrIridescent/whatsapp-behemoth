# 🚀 WhatsApp Behemoth v4.0.0
### *The Ultimate AI-Powered Marketing Workstation*

**WhatsApp Behemoth** is a high-performance, cross-platform desktop suite architected for professional outreach, multi-account dominance, and autonomous AI-driven engagement.

## 🌟 Key Features

- **🧠 AI Intelligence Core**: Real-time chat summarization and reply generation using OpenAI, Anthropic, or Google Gemini.
- **⚡ Marketing HUB**: Broadcast messages to thousands of recipients with integrated CSV/TXT support.
- **🛡️ Stealth Engine**: Advanced anti-detection protocols, randomized delays, and unique User-Agent spoofing per station.
- **👥 Group Extractor**: Harvest participant data from any group instantly for your marketing campaigns.
- **🗂️ Multi-Station Architecture**: Run 5+ isolated WhatsApp and WhatsApp Business sessions simultaneously.
- **🕵️ Privacy Mode**: Automatic UI blurring and stealth networking to hide your online status.

## 🛠️ Installation & Deployment

### **Quick Install**
Download the latest installer for your OS from the [Releases Page](https://github.com/MrIridescent/whatsapp-behemoth/releases).

*   **Debian/Kali**: `sudo dpkg -i WhatsApp-Behemoth-4.0.0-amd64.deb`
*   **Windows**: Run `WhatsApp-Behemoth-4.0.0-x64.exe`
*   **macOS**: Extract `WhatsApp-Behemoth-4.0.0-x64.zip` and move the `.app` to Applications.
